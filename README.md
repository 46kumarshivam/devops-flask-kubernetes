 Project Overview



This project demonstrates a \*\*full DevOps workflow\*\* with a Flask application deployed locally using:



 \*\*Docker\*\* containers

 \*\*Kubernetes\*\* deployment with Minikube

 \*\*NodePort Service\*\* for accessing the app

 Ready for CI/CD integration with Jenkins and monitoring with Prometheus/Grafana

\## Tech Stack



| Technology      | Purpose                          |
